# Gokarna Trip Expense Tracker

A beautiful, mobile-first expense tracker for your Gokarna trip.

## Features
- **Track Expenses**: Add expenses with amount, category, and comments.
- **Visual Overview**: See total spent and recent transactions.
- **Persistent**: Data is saved to your device automatically.
- **Premium Design**: Dark mode with glassmorphism and smooth animations.

## How to Deploy on Vercel

1. **Push to GitHub**:
   - Initialize a git repository (if not already):
     ```bash
     git init
     git add .
     git commit -m "Initial commit"
     ```
   - Create a new repository on GitHub and push this code.

2. **Deploy**:
   - Go to [Vercel](https://vercel.com).
   - Click "Add New Project".
   - Import your GitHub repository.
   - Click "Deploy".

## Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
